Stress Test

This small app is designed to test the effects of different design decisions on how quickly texts are rendered in the browser. Select the feature you ºd like to test, then click Run Test. The time it took to execute the function will display below, and the results of the test will be stringified and stored in local storage along with the test settings for that particular test run. The test data is formatted like the following sample:

      localStorage.testResults = [                // An array of test results
        {                                         // A single test result, stored as a JSON object
          "date": "2015-02-11T07:07:04.762Z",     // The date the test was run
          "renderMethod": "with DOM IDs",         // The render method that was used for this test
          "browser": "Chrome/40.0.2214.111",      // The browser the test was run in (shortened here for display purposes)
          "speed": 2.34                           // The time it took the function to execute, in seconds
        }
      ];
